|           |            |                                                              |

### 支付状态(PayStatusEnum)

| 编码     | 类型     | 备注                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| wait     | 待支付   | 网关支付创建的订单通常默认是这个状态，默认未选择支付通道和支付方式 |
| progress | 支付中   | 已经选择了支付通道和支付方式                                 |
| success  | 成功     |                                                              |
| close    | 支付关闭 |                                                              |
| cancel   | 支付撤销 |                                                              |
| timeout  | 支付超时 | 订单到了超时时间, 被手动设置订单为这个状态，通常作为中间流转状态，可以视为支付关闭 |
| fail     | 失败     |                                                              |

### 退款状态(PayRefundStatusEnum)
